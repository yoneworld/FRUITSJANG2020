# FruitsJang2023
- For SHARP PC-G850 Series
- FruitJang2023.bas

# 遊び方
## キー操作
| キー |  意味 |
| --- | ---- |
|  /  |  あがり  |
|  4,6  | フルーツ選択  |
|  5 |  捨てる  |

## スコア
| 役 | 意味 | pts |
| --- | --- | --- |
| フルーツペア | 4組のペア | 1000 |
|  フルーツカルテット |  4個ずつのペア  | 3000 |
|  シックスフルーツ | 2個と6個のペア | 5000 |
|  オールセイム | すべて同じ  |  10000 | 
|  フルーツアラカルト | 8種すべてそろう  | 1000 |
|  ファーストメイク | 1回目であがる  | +5000 |
|  ボーナスフルーツ | 毎回変わる | +500 x フルーツ数 |

# CharacterCode/改行コードなど
```
SHIFT_JIS
CRLF
ファイル末尾にCTRL+Zをつけて実機に転送する
```
